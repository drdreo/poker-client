Frontend made with Angular 11

Server made with Nest.js

# Poker

## Running the client
Run these commands inside the `client` folder
```bash
# development
npm run start

# HMR mode
npm run start:hmr

# production build
npm run build
```

## Running the server
Run these commands inside the `root` folder

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod

# production build
npm run build
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
