# Poker

## Running the client
Run these commands inside the `client` folder
```bash
# development
npm run start

# HMR mode
npm run start:hmr

# production build
npm run build
```

## Running the server
Run these commands inside the `server` folder

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod

# production build
npm run build
```

## Test

Inside root: 
```bash
# unit tests
npm run test
```

Inside client: 
```bash
# unit tests 
npm run test
# e2e tests
npm run cypress
```
